Network Settings

If you are working inside the BBC Network set this value to "BBC Reith Proxy"
If you are working with no internet connection and running the application locally set the value to "Use locally cached files only"

Controls the amount of time data files are cached server side on the file system
It is recommended to use the highest value possible to reduce load on the OPTA api.

Using the browser cache will improve the performance of the application when downloading files.
